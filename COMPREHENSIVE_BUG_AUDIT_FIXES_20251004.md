
**File:** `api/risk_assessment_endpoints.py` (line 361-371)  
**Severity:** üî¥ CRITICAL  
**Impact:** AttributeError on None objects causing 500 errors

**Problem:**

**Fix:**
```python

    raise HTTPException(status_code=400, detail="Failed to process image")
```
